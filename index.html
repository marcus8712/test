Hoiio Design Standards
A carefully crafted design guideline developed by Hoiio UX Team.
Color

The Hoiio palette includes 6 main colors which represents 6 states applied to mostly web applications.

primary
info
success
warning
error
pending
Font

Hoiio uses the Poppins family, which includes five weights, from Light through Bold. Each font has 1014 glyphs, including all of the unique conjunct forms necessary for typesetting Indian languages like Hindi, Marathi, Nepali, etc.

Icon

Currently it has 409 icons, including all icons of Entypo and several custom icons.

These icons are splitted into 14 categories: action, communication, content, device, direction, document, editor, media, navigation, notification, people, social, thing, and toggle.
